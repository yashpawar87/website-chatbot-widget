Test the RAG Chatbot Widget

You should see a floating chat bubble in the bottom right corner of this page. Click it to open the chat interface.

How it works:

  • The script tag below injects widget.js from your Next.js server (running on port 3000).
  • The data-api-url attribute tells the widget where your FastAPI backend is running (port 8000).
  • Once clicked, the widget opens an iframe pointing to the Next.js /widget-iframe route.
